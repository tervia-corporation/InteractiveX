InteractiveX Engine

Direct3D 9 engine for Windows software, tools and games.

InteractiveX is written in C/C++ and focuses on practical systems for rendering, UI, audio, input, scenes and media playback, with a cleaner portal-style presentation on this site.

Read documentation Download r0.2 Download demo
Core systems
  • Scenes, renderer and GUI drawing
  • Input, gamepad, timer and camera
  • Entities, tilemaps and animation
  • Transitions, save/load and helpers
Media
  • Music and sound effects
  • PNG image loading
  • MPEG-1 video playback
  • Splash screen support
Interface
  • Buttons, labels, panels and sliders
  • Layout helpers for UI composition
  • Gradients, borders and text drawing
  • Simple integration into engine scenes
